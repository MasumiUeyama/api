Methods For: MultiplePrinter (NewCreator for jp.ac.oit.igakilab.dwr.multiple.MultiplePrinter)

To use this class in your javascript you will need the following script includes:

  <script type='text/javascript' src='/multiple-dwr/dwr/engine.js'></script>
  <script type='text/javascript' src='/multiple-dwr/dwr/interface/MultiplePrinter.js'></script>

In addition there is an optional utility script:

  <script type='text/javascript' src='/multiple-dwr/dwr/util.js'></script>

Replies from DWR are shown with a yellow background if they are simple or in an alert box otherwise.
